# 📊 Alura Store LATAM - Análisis de Datos

Este proyecto realiza un **análisis exploratorio de datos (EDA)** sobre las ventas de varias tiendas de **Alura Store en Latinoamérica**.  
El objetivo principal es **analizar el desempeño de las tiendas**, identificar patrones de ventas y obtener insights que apoyen la **toma de decisiones del negocio**.

El análisis se desarrolló utilizando **Python y Pandas en un notebook de Jupyter**, trabajando con múltiples datasets correspondientes a diferentes tiendas.

---

# 🚀 Objetivos del Proyecto

Analizar la información de ventas de varias tiendas para responder preguntas clave del negocio:

- 📈 ¿Cuál es la **facturación total de cada tienda**?
- 🛍️ ¿Qué **categorías de productos** venden más?
- ⭐ ¿Cuál es la **calificación promedio de los clientes**?
- 📦 ¿Cuáles son los **productos más y menos vendidos**?
- 🚚 ¿Cuál es el **costo promedio de envío por tienda**?

A partir de estos análisis se pueden identificar **oportunidades de mejora y optimización comercial**.

---

# 🗂️ Dataset

Los datos utilizados provienen de diferentes archivos CSV correspondientes a cuatro tiendas:

- `tienda_1.csv`
- `tienda_2.csv`
- `tienda_3.csv`
- `tienda_4.csv`

Cada dataset contiene información como:

- Producto
- Categoría
- Precio
- Costo de envío
- Calificación del cliente
- Cantidad vendida
- Ubicación de la compra

Estos datos fueron cargados directamente desde un repositorio público y analizados con **Pandas**.

# 📊 Análisis Realizados

## 1️⃣ Análisis de Facturación

Se calculó la **facturación total de cada tienda** para identificar cuál tiene el mayor volumen de ingresos.

Este análisis permite:

- Comparar el desempeño financiero entre tiendas
- Identificar tiendas con mayor impacto en el negocio

---

## 2️⃣ Ventas por Categoría

Se analizaron las ventas agrupadas por **categoría de producto**.

Esto permite identificar:

- Las categorías más populares
- Tendencias de compra de los clientes
- Oportunidades de expansión en productos con alta demanda

---

## 3️⃣ Calificación Promedio de la Tienda

Se calculó la **calificación promedio otorgada por los clientes**.

Este indicador es importante porque:

- Refleja la satisfacción del cliente
- Permite comparar la experiencia de compra entre tiendas
- Ayuda a detectar posibles problemas en el servicio

---

## 4️⃣ Productos Más y Menos Vendidos

Se identificaron:

- 📈 **Productos más vendidos**
- 📉 **Productos con menor demanda**

Esto ayuda a:

- Optimizar inventarios
- Promocionar productos con baja rotación
- Potenciar los productos con mayor éxito comercial

---

## 5️⃣ Costo Promedio de Envío por Tienda

Se calculó el **promedio del costo de envío** para cada tienda.

Este análisis permite:

- Evaluar el impacto logístico
- Comparar costos de distribución
- Identificar oportunidades de optimización en envíos

---

# 📈 Posibles Insights de Negocio

A partir del análisis se pueden obtener conclusiones como:

- Identificar **la tienda con mayor rendimiento financiero**
- Detectar **categorías más rentables**
- Reconocer **productos líderes en ventas**
- Analizar el **impacto de los costos logísticos**
- Evaluar la **satisfacción del cliente**

Estos insights pueden utilizarse para **optimizar la estrategia comercial de Alura Store**.
